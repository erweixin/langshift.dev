# 模块 03: 控制流和循环

在本模块中，我们将探索 Java 的控制流机制，包括条件语句、循环和分支。虽然 Java 和 JavaScript 共享许多相似的控制结构，但存在重要的语法和行为差异，JavaScript 开发者需要理解这些差异。

## 学习目标

通过本模块的学习，你将能够：
- 理解 Java 的条件语句及其语法
- 掌握 Java 中不同类型的循环
- 比较 Java 和 JavaScript 的控制流模式
- 有效使用 break 和 continue 语句
- 应用 Java 中控制流的最佳实践

## 条件语句

### 1. If-Else 语句

两种语言都有相似的 if-else 语法，但 Java 需要显式的布尔表达式。

<UniversalEditor title="If-Else 语句对比" compare={true}>
```javascript !! js
// JavaScript - If-else 语句

let score = 85;
let name = "John";

// 简单 if 语句
if (age >= 18) {
    console.log("Adult");
}

// If-else 语句
if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// If-else if-else 链
if (score >= 90) {
    console.log("Excellent");
} else if (score >= 80) {
    console.log("Good");
} else if (score >= 70) {
    console.log("Average");
} else {
    console.log("Needs improvement");
}

// 真值/假值
if (name) {
    console.log("Name exists");
}

if (score) {
    console.log("Score is truthy");
}

// 三元运算符
let status = age >= 18 ? "Adult" : "Minor";
console.log(status);
```

```java !! java
            System.out.println("Adult");

        // If-else 语句
        if (age >= 18) {
            System.out.println("Adult");
        } else {
            System.out.println("Minor");
        }
        
        // If-else if-else 链
        if (score >= 90) {
            System.out.println("Excellent");
        } else if (score >= 80) {
            System.out.println("Good");
        } else if (score >= 70) {
            System.out.println("Average");
        } else {
            System.out.println("Needs improvement");

                System.out.println("Adult with poor score");
            }
        } else {
            System.out.println("Minor");
        }
    }
```

```java !! java
    public static void main(String[] args) {
        int day = 3;

        switch (day) {
            case 1:
                System.out.println("Monday");
                break;
            case 2:
                System.out.println("Tuesday");
                break;
            case 3:
                System.out.println("Wednesday");
                break;

                break;
            case 5:
                System.out.println("Friday");
                break;
            default:
                System.out.println("Weekend");
        }
        
        // 带字符串的 switch（Java 7+）
        switch (fruit) {
            case "apple":
                System.out.println("Red fruit");
                break;
            case "banana":
                System.out.println("Yellow fruit");
                break;
            case "orange":
                System.out.println("Orange fruit");
                break;
            default:
                System.out.println("Unknown fruit");
        }
        
        // 没有 break 的 switch（贯穿）
        switch (day) {
            case 1:
            case 2:
            case 3:
            case 4:
            case 5:
                System.out.println("Weekday");
                break;
            case 6:
            case 7:
                System.out.println("Weekend");
                break;
            default:
                System.out.println("Invalid day");
        }
        
        // 增强 switch 语句（Java 14+）
        String dayName = switch (day) {
            case 1 -> "Monday";
            case 2 -> "Tuesday";
            case 3 -> "Wednesday";
            case 4 -> "Thursday";
            case 5 -> "Friday";
            case 6, 7 -> "Weekend";
            default -> "Invalid day";
        };
        System.out.println("Day name: " + dayName);
        
        // 带多语句的 switch 表达式
        String result = switch (day) {
            case 1, 2, 3, 4, 5 -> {
                String weekday = "Weekday";
                yield weekday + " - Day " + day;
            }
            case 6, 7 -> {
                String weekend = "Weekend";
                yield weekend + " - Day " + day;
            }
            default -> "Invalid day";
        };
        System.out.println(result);
    }
}
```

## 循环

### 1. For 循环

Java 提供几种类型的 for 循环，各有不同的用例。

<UniversalEditor title="For 循环对比" compare={true}>
```javascript !! js
// JavaScript - For 循环
// 传统 for 循环
for (let i = 0; i < 5; i++) {
    console.log("Count: " + i);
}

// For...of 循环（遍历值）
const fruits = ["apple", "banana", "orange"];
for (const fruit of fruits) {
    console.log(fruit);
}

// For...in 循环（遍历键）
const person = {name: "John", age: 30, city: "NYC"};
for (const key in person) {
    console.log(key + ": " + person[key]);
}

// 数组 forEach 方法
fruits.forEach((fruit, index) => {
    console.log(`Fruit ${index}: ${fruit}`);
});

// 带 break 的 for 循环
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;
    }
    console.log(i);
}

// 带 continue 的 for 循环
for (let i = 0; i < 10; i++) {
    if (i % 2 === 0) {
        continue;
    }
    console.log("Odd: " + i);
}
```

```java !! java
// Java - For 循环
import java.util.Arrays;
import java.util.List;

public class ForLoops {
    public static void main(String[] args) {
        // 传统 for 循环
        for (int i = 0; i < 5; i++) {
            System.out.println("Count: " + i);
        }
        
        // 增强 for 循环（for-each）- 遍历值
        String[] fruits = {"apple", "banana", "orange"};
        for (String fruit : fruits) {
            System.out.println(fruit);
        }
        
        // 带 List 的增强 for 循环
        List<String> fruitList = Arrays.asList("apple", "banana", "orange");
        for (String fruit : fruitList) {
            System.out.println(fruit);
        }
        
        // 带数组的传统 for 循环
        for (int i = 0; i < fruits.length; i++) {
            System.out.println("Fruit " + i + ": " + fruits[i]);
        }
        
        // 带 break 的 for 循环
        for (int i = 0; i < 10; i++) {
            if (i == 5) {
                break;
            }
            System.out.println(i);
        }
        
        // 带 continue 的 for 循环
        for (int i = 0; i < 10; i++) {
            if (i % 2 == 0) {
                continue;
            }
            System.out.println("Odd: " + i);
        }
        
        // for 循环中的多个变量
        for (int i = 0, j = 10; i < j; i++, j--) {
            System.out.println("i: " + i + ", j: " + j);
        }
        
        // 带步长的 for 循环
        for (int i = 0; i < 20; i += 2) {
            System.out.println("Even: " + i);
        }
        
        // Lambda forEach（Java 8+）
        fruitList.forEach(fruit -> System.out.println("Lambda: " + fruit));
    }
}
```

```java !! java
// Java - 嵌套控制结构
import java.util.Arrays;
import java.util.List;

public class NestedControlStructures {
    public static void main(String[] args) {
        // Student 类定义
        class Student {
            String name;
            int age;
            int grade;

                this.name = name;
                this.age = age;
                this.grade = grade;
            }
        }

            new Student("Alice", 20, 85),
            new Student("Bob", 19, 92),
            new Student("Charlie", 21, 78)
        );
        
        // 嵌套循环和条件
        for (Student student : students) {
            if (student.age >= 18) {
                if (student.grade >= 90) {
                    System.out.println(student.name + " is an excellent adult student");
                } else if (student.grade >= 80) {
                    System.out.println(student.name + " is a good adult student");
                } else {
                    System.out.println(student.name + " is an adult student needing improvement");
                }
            } else {
                System.out.println(student.name + " is a minor student");
            }
        }
        
        // 复杂嵌套条件
        processUser(new User("John", 25, "john@example.com"));
        processUser(new User("Jane", 16, null));
        processUser(new User(null, 30, "user@example.com"));
        processUser(null);
    }
    
    static void processUser(User user) {
        if (user != null) {
            if (user.name != null && !user.name.trim().isEmpty()) {
                if (user.age >= 18) {

                        System.out.println("Valid adult user without email");
                    }

                    System.out.println("Minor user");
                }
            } else {
                System.out.println("User without name");
            }

        }
    }

        String email;
```

### 2. 早期返回和守卫子句

<UniversalEditor title="早期返回和守卫子句" compare={true}>
```javascript !! js
// JavaScript - 早期返回和守卫子句
function processUser(user) {
    // 守卫子句 - 无效情况的早期返回
    if (!user) {
        return "No user provided";
    }
    
    if (!user.name) {
        return "User without name";
    }
    
    if (user.age < 18) {
        return "Minor user";
    }
    
    if (!user.email) {
        return "Adult user without email";
    }
    
    // 有效用户的主要逻辑
    return "Valid adult user with email: " + user.email;
}

function calculateGrade(score) {
    // 无效分数的守卫子句
    if (score < 0 || score > 100) {
        return "Invalid score";
    }
    
    // 主要逻辑
    if (score >= 90) return "A";
    if (score >= 80) return "B";
    if (score >= 70) return "C";
    if (score >= 60) return "D";
    return "F";
}

// 使用
console.log(processUser({name: "John", age: 25, email: "john@example.com"}));
console.log(processUser(null));
console.log(calculateGrade(85));
console.log(calculateGrade(150));
```

```java !! java
// Java - Switch 表达式模式（Java 14+）
public class SwitchExpressions {
    public static void main(String[] args) {
        // 返回值的 switch 表达式
        int day = 3;
        String dayAction = switch (day) {
            case 1, 2, 3, 4, 5 -> "Work";
            case 6, 7 -> "Rest";
            default -> "Invalid day";
        };
        System.out.println("Day " + day + ": " + dayAction);
        
        // 带复杂逻辑的 switch 表达式
        int score = 85;
        String grade = switch (score) {
            case 90, 91, 92, 93, 94, 95, 96, 97, 98, 99, 100 -> "A";
            case 80, 81, 82, 83, 84, 85, 86, 87, 88, 89 -> "B";
            case 70, 71, 72, 73, 74, 75, 76, 77, 78, 79 -> "C";
            case 60, 61, 62, 63, 64, 65, 66, 67, 68, 69 -> "D";
            default -> "F";
        };
        System.out.println("Score " + score + ": " + grade);
        
        // 带 yield 的复杂逻辑 switch 表达式
        String result = switch (day) {
            case 1, 2, 3, 4, 5 -> {
                String workDay = "Work day";
                yield workDay + " - Day " + day;
            }
            case 6, 7 -> {
                String weekend = "Weekend";
                yield weekend + " - Day " + day;
            }
            default -> "Invalid day";
        };
        System.out.println(result);
        
        // 带模式匹配的 switch 表达式（Java 17+）
        Object obj = "Hello";
        String type = switch (obj) {
            case String s -> "String: " + s;
            case Integer i -> "Integer: " + i;
            case Double d -> "Double: " + d;
            case null -> "Null";
            default -> "Unknown type";
        };
        System.out.println(type);
    }
}
```
