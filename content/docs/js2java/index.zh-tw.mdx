# JavaScript → Java

從 JavaScript 開發者視角掌握 Java 程式語言。本課程全面涵蓋物件導向程式設計、企業級開發、Spring 框架、並發程式設計、JVM 優化和現代 Java 特性。

## 為什麼學習 Java？

Java 是企業級開發中最受歡迎的程式語言之一，以其以下特點著稱：

- **物件導向設計**：強大的基於類的繼承和多態
- **企業級特性**：大規模應用的廣泛生態系統
- **Spring 框架**：企業級 Java 開發的主導框架
- **並發程式設計**：內建的多執行緒和並行處理支援
- **JVM 性能**：優化的執行時和垃圾回收
- **跨平台**：一次編寫，到處執行的理念

## 你將學到什麼

### 核心 Java 概念
- 物件導向程式設計原則
- 強型別和型別安全
- 異常處理和錯誤管理
- 泛型和集合框架
- 並發程式設計和多執行緒

### 企業級開發
- Spring 框架基礎
- 依賴注入和控制反轉
- RESTful API 開發
- 資料庫整合（JPA）
- 微服務架構

### 現代 Java 特性
- Lambda 表達式和函數式程式設計
- Stream API 資料處理
- Optional 類的空安全
- 模組系統（Java 9+）
- 性能優化技術

## 學習路徑

### 基礎階段（模組 0-5）
從 Java 基礎、語法對比和核心概念開始。

### 物件導向程式設計（模組 6-10）
掌握類、繼承、介面和進階 OOP 概念。

### 進階特性（模組 11-14）
學習集合、並發和 Spring 框架。

### 實戰應用（模組 15-19）
建構實際專案並優化性能。

## 前置要求

- 基礎 JavaScript 知識
- 程式設計基礎概念理解
- Web 開發概念熟悉

## 課程特色

- **互動式程式碼範例**：直接在瀏覽器中執行 Java 程式碼
- **對比學習**：查看 JavaScript 和 Java 實作對比
- **實戰專案**：建構企業級應用
- **性能分析**：理解 JVM 優化
- **最佳實踐**：學習 Java 編碼標準和模式
