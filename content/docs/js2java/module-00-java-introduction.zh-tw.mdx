# Java 語言介紹

歡迎來到 Java 程式設計語言的世界！作為 JavaScript 開發者，你將發現 Java 提供了一個全新的程式設計範式，特別是在物件導向程式設計和企業級開發方面。

## Java 語言歷史

Java 由 Sun Microsystems 的 James Gosling 團隊在 1995 年創建，最初被稱為 Oak。它的設計目標是：

- **Write Once, Run Anywhere (WORA)**: 一次編寫，到處運行
- **物件導向**: 完全基於物件的設計
- **安全性**: 內建的安全機制
- **網路程式設計**: 原生支援網路應用開發

<UniversalEditor title="Hello World 對比">
```javascript !! js
// JavaScript - 簡單的 Hello World
console.log("Hello, World!");

// 在瀏覽器中直接運行
// 無需編譯，解釋執行
```

```java !! java
// Java - Hello World 程式
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

// 需要編譯為位元組碼，然後在 JVM 上運行
// javac HelloWorld.java
```

## Java vs JavaScript 設計哲學對比

### JavaScript 的設計哲學
- **動態型別**: 變數型別在執行時確定
- **原型繼承**: 基於原型的物件系統
- **函數式程式設計**: 函數是一等公民
- **解釋執行**: 直接解釋執行，無需編譯

### Java 的設計哲學
- **靜態型別**: 編譯時確定型別
- **類別繼承**: 基於類別的物件導向系統
- **強型別**: 嚴格的型別檢查
- **編譯執行**: 編譯為位元組碼，在 JVM 上運行

<UniversalEditor title="型別系統對比">
```javascript !! js
// JavaScript - 動態型別
let name = "John";           // 字串
name = 42;                   // 數字
name = { age: 25 };          // 物件
name = [1, 2, 3];            // 陣列
name = function() {};        // 函數

// 型別在執行時確定，靈活但可能導致錯誤
```

```java !! java
// Java - 靜態型別
String name = "John";        // 字串
// name = 42;                // 編譯錯誤！
// name = new Object();      // 編譯錯誤！

int age = 25;               // 整數
double price = 19.99;       // 浮點數
boolean isActive = true;    // 布林值

// 型別在編譯時確定，安全但需要顯式宣告
```
</UniversalEditor>

## Java 的優勢

### 1. 企業級開發
Java 在企業級應用開發中佔據主導地位：

<UniversalEditor title="企業級應用範例">
```javascript !! js
// JavaScript - 簡單的 Web 服務
const express = require('express');
const app = express();

app.get('/api/users', (req, res) => {
    res.json({ users: [] });
});

app.listen(3000, () => {
    console.log('Server running on port 3000');
});
```

```java !! java
// Java - Spring Boot 企業級應用
@SpringBootApplication
@RestController
public class UserController {
    
    @Autowired
    private UserService userService;
    
    @GetMapping("/api/users")
    public ResponseEntity<List<User>> getUsers() {
        List<User> users = userService.getAllUsers();
        return ResponseEntity.ok(users);
    }
    
    public static void main(String[] args) {
        SpringApplication.run(UserController.class, args);
    }
}

// 包含依賴注入、事務管理、安全控制等企業級特性
```
</UniversalEditor>

### 2. 效能優勢
- **JIT 編譯**: 即時編譯最佳化
- **垃圾回收**: 自動記憶體管理
- **多執行緒**: 原生並發支援
- **最佳化**: 成熟的效能最佳化工具

### 3. 生態系統
- **Spring 框架**: 企業級開發框架
- **Maven/Gradle**: 建置工具
- **JUnit**: 測試框架
- **IDE 支援**: IntelliJ IDEA, Eclipse

## 開發環境搭建

### 1. 安裝 JDK (Java Development Kit)

#### Windows
1. 下載 OpenJDK 或 Oracle JDK
2. 執行安裝程式
3. 設定環境變數 `JAVA_HOME`

#### macOS
```bash
# 使用 Homebrew
brew install openjdk@17

# 或使用 SDKMAN
curl -s "https://get.sdkman.io" | bash
sdk install java 17.0.2-open
```

#### Linux
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# CentOS/RHEL
sudo yum install java-17-openjdk-devel
```

## Java 平台架構

### JVM (Java Virtual Machine)
- **位元組碼執行**: 將 Java 位元組碼轉換為機器碼
- **跨平台**: 同一份位元組碼在不同平台運行
- **記憶體管理**: 自動垃圾回收
- **效能最佳化**: JIT 編譯和最佳化

<UniversalEditor title="JVM 架構理解">
```java !! java
// Java 原始碼
public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

// 編譯後生成位元組碼
// javac Calculator.java 生成 Calculator.class
// 位元組碼可以在任何有 JVM 的平台上運行
```

```javascript !! js
// JavaScript 直接解釋執行
function add(a, b) {
    return a + b;
}

// 不同瀏覽器的 JavaScript 引擎實現不同
// V8 (Chrome), SpiderMonkey (Firefox), JavaScriptCore (Safari)
```
</UniversalEditor>

## Java 版本和特性

### Java 8 (LTS)
- Lambda 表達式
- Stream API
- Optional 類別
- 新的日期時間 API

### Java 11 (LTS)
- HTTP Client
- 區域變數型別推斷
- 字串方法增強

### Java 17 (LTS)
- 密封類別
- 模式匹配
- 文字區塊
- 記錄類別

<UniversalEditor title="現代 Java 特性">
```java !! java
// Java 8+ Lambda 表達式
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
names.forEach(name -> System.out.println("Hello, " + name));

// Java 10+ 區域變數型別推斷
var message = "Hello, World!";
var numbers = List.of(1, 2, 3, 4, 5);

// Java 14+ 記錄類別
record Person(String name, int age) {}

// Java 15+ 文字區塊
String sql = """
    SELECT id, name, email 
    FROM users 
    WHERE active = true
    """;
```

```javascript !! js
// JavaScript 對應特性
const names = ["Alice", "Bob", "Charlie"];
names.forEach(name => console.log(`Hello, ${name}`));

// 動態型別，無需宣告
const message = "Hello, World!";
const numbers = [1, 2, 3, 4, 5];

// 物件字面量
const person = { name: "John", age: 30 };

// 模板字串
const sql = `
    SELECT id, name, email 
    FROM users 
    WHERE active = true
`;
```
