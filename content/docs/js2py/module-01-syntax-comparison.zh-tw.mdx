## 1. 引言

### 為什麼需要語法對比？

作為 JavaScript 開發者，學習 Python 最大的挑戰之一就是適應不同的語法風格。通過系統性的語法對比，我們可以：

- 快速理解 Python 的語法規則
- 避免常見的語法錯誤
- 建立兩種語言之間的概念映射
- 提高學習效率

> 💡 **學習策略**：將 Python 語法視為 JavaScript 的"方言"，而不是完全陌生的語言

**關鍵差異對比**

| 特性 | JavaScript | Python | 說明 |
|------|------------|--------|------|
| **聲明方式** | `let/const/var` | 直接賦值 | Python 無需聲明關鍵字 |
| **變量名** | camelCase | snake_case | Python 推薦下劃線命名 |
| **常量** | `const` | 約定俗成（全大寫） | Python 沒有真正的常量 |
| **字符串格式化** | 模板字符串 | f-string | 兩者都支持插值 |

### 2.3 作用域對比

作用域是編程中非常重要的概念，它決定了變量在代碼中的可見性和生命週期。JavaScript 和 Python 在作用域處理上有一些重要差異。

#### 2.3.1 變量作用域

<PythonEditor title="作用域示例" compare={true}>
```javascript !! js
// 全局作用域
let globalVar = "我是全局變量";

function testScope() {
    // 函數作用域
    let functionVar = "我是函數內變量";
    
    if (true) {
        // 塊級作用域（ES6+）
        let blockVar = "我是塊級變量";
        console.log("塊內訪問:", blockVar);
        console.log("塊內訪問函數變量:", functionVar);
        console.log("塊內訪問全局變量:", globalVar);
    }
    
    // 塊外無法訪問塊級變量
    // console.log(blockVar); // 錯誤！
    console.log("函數內訪問函數變量:", functionVar);
    console.log("函數內訪問全局變量:", globalVar);
}

testScope();
console.log("全局訪問全局變量:", globalVar);
// console.log(functionVar); // 錯誤！
```

```python !! py
# 全局作用域
global_var = "我是全局變量"

def test_scope():
    # 函數作用域
    function_var = "我是函數內變量"
    
    if True:
        # Python 沒有塊級作用域！
        block_var = "我是塊級變量"
        print("塊內訪問:", block_var)
        print("塊內訪問函數變量:", function_var)
        print("塊內訪問全局變量:", global_var)
    
    # 塊外仍然可以訪問塊級變量
    print("函數內訪問塊級變量:", block_var)
    print("函數內訪問函數變量:", function_var)
    print("函數內訪問全局變量:", global_var)

test_scope()
print("全局訪問全局變量:", global_var)
# print(function_var)  # 錯誤！
```
</PythonEditor>

```python !! py
    counter = 0  # 創建局部變量
    counter += 1

increment()
increment()
increment_local()
```

#### 2.3.3 閉包對比

**JavaScript 閉包**

<PythonEditor title="閉包對比" compare={true}>
```javascript !! js
// JavaScript 閉包
function createCounter() {
    let count = 0;
    
    return {
        increment: function() {
            count++;
            return count;
        },
        getCount: function() {
            return count;
        }
    };
}

const counter1 = createCounter();
const counter2 = createCounter();

console.log(counter1.increment()); // 1
console.log(counter1.increment()); // 2
console.log(counter2.increment()); // 1
console.log(counter1.getCount());  // 2
console.log(counter2.getCount());  // 1
```

```python !! py
# Python 閉包
def create_counter():
    count = 0
    
    def increment():
        nonlocal count  # 聲明使用外層變量
        count += 1
        return count
    
    def get_count():
        return count
    
    return {
        'increment': increment,
        'get_count': get_count
    }

counter1 = create_counter()
counter2 = create_counter()

print(counter1['increment']())  # 1
print(counter1['increment']())  # 2
print(counter2['increment']())  # 1
print(counter1['get_count']())  # 2
print(counter2['get_count']())  # 1
```
</PythonEditor>

**作用域關鍵差異總結**

| 特性 | JavaScript | Python | 說明 |
|------|------------|--------|------|
| **塊級作用域** | ES6+ 支持 | 不支持 | Python 的 if/for 不創建新作用域 |
| **全局變量聲明** | 直接修改 | 需要 `global` 關鍵字 | Python 更嚴格 |
| **閉包變量** | 自動捕獲 | 需要 `nonlocal` 關鍵字 | Python 需要顯式聲明 |
| **變量提升** | 存在 | 不存在 | Python 更直觀 |

```python !! py
# 函數定義
def greet(name):
    return f"你好，{name}！"

# 默認參數
def greet2(name="世界"):
    return f"你好，{name}！"

# 類型註解（Python 3.5+）
def greet3(name: str) -> str:
    return f"你好，{name}！"

# 多參數
def greet4(name, age):

def greet6(person):

    age = person["age"]

print(greet2())
print(greet6({"name": "李四", "age": 25}))
```

## 5. 錯誤處理對比

### 5.1 異常處理

<PythonEditor title="異常處理對比" compare={true}>
```javascript !! js
// 基本錯誤處理
try {
    const result = 10 / 0;
    console.log(result);
} catch (error) {
    console.error("發生錯誤:", error.message);
} finally {
    console.log("總是執行");
}

// 自定義錯誤
class CustomError extends Error {
    constructor(message) {
        super(message);
        this.name = "CustomError";
    }
}

// 拋出錯誤
function divide(a, b) {
    if (b === 0) {
        throw new CustomError("除數不能為零");
    }
    return a / b;
}

try {
    console.log(divide(10, 0));
} catch (error) {
    console.error("除法錯誤:", error.message);
}
```

```python !! py
# 基本錯誤處理
try:
    result = 10 / 0
    print(result)
except ZeroDivisionError as e:
    print(f"發生錯誤: {e}")
except Exception as e:
    print(f"其他錯誤: {e}")
else:
    print("沒有異常時執行")
finally:
    print("總是執行")

# 自定義異常
class CustomError(Exception):
    def __init__(self, message):
        self.message = message
        super().__init__(self.message)
```

**解釋：**

- 內層函數可以訪問外層變量，但同名變量會創建新的局部變量

## 8. 總結

### 關鍵差異總結

| 方面 | JavaScript | Python | 學習建議 |
|------|------------|--------|----------|
| **語法風格** | 大括號 + 分號 | 縮進敏感 | 注意 Python 的縮進規則 |
| **變量聲明** | 需要關鍵字 | 直接賦值 | 適應 Python 的簡潔性 |
| **命名規範** | camelCase | snake_case | 遵循 Python 的命名約定 |
| **字符串格式化** | 模板字符串 | f-string | 兩者都很直觀 |
| **循環語法** | for/while | for/while | Python 的 for 更簡潔 |
| **函數定義** | function/箭頭函數 | def | 語法差異較大 |
| **錯誤處理** | try-catch | try-except | 概念相似，語法不同 |
| **作用域** | 塊級作用域 | 函數作用域 | 注意 Python 的全局變量聲明 |
