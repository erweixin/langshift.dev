## 1. 引言

### Python: 數據處理的瑞士軍刀

雖然 JavaScript (特別是 Node.js) 能夠處理文件和執行腳本，但 Python 在數據處理、科學計算和自動化任務方面擁有更強大、更成熟的生態系統。對於許多開發者來說，這正是他們學習 Python 的主要原因之一。

本模塊將向你展示 Python 如何輕鬆應對各種數據處理挑戰，從簡單的文件 I/O 到複雜的數據分析。

**核心概念類比**

| Python | JavaScript | 主要功能 |
| --- | --- | --- |
| **內置文件操作** | `fs` 模塊 | 文件讀寫 |
| **`pathlib`** | `path` 模塊 | 跨平臺路徑操作 |
| **`re` 模塊** | `RegExp` 對象 | 正則表達式 |
| **`pandas`** | `lodash`, `danfo.js` | 高性能數據分析 |
| **`csv` 模塊** | `csv-parser`, `papaparse` | CSV 文件讀寫 |

> 💡 **學習策略**：將 Python 的這些工具視為你現有 JavaScript/Node.js 工作流的“超級升級版”。你會發現 Python 在處理結構化數據和大型數據集時更加得心應手。

## 2. 文件操作

Python 提供了非常簡潔直觀的文件操作語法。

### 2.1 讀寫文件

<PythonEditor title="文件讀寫對比" compare={true} canRun={false}>
```javascript !! js
// JavaScript (Node.js)
const fs = require('fs');

// 寫文件
fs.writeFileSync('hello.txt', 'Hello from Node.js!');

// 讀文件
const content = fs.readFileSync('hello.txt', 'utf8');
console.log(content);

// 異步讀寫
fs.writeFile('hello_async.txt', 'Async hello!', (err) => {
  if (err) throw err;
  fs.readFile('hello_async.txt', 'utf8', (err, data) => {
    if (err) throw err;
    console.log(data);
  });
});
```

```python !! py
# Python
# 寫文件
with open("hello.txt", "w", encoding="utf-8") as f:
    f.write("Hello from Python!")

# 讀文件
with open("hello.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# pathlib (更現代的方式)
from pathlib import Path

Path("hello_pathlib.txt").write_text("Hello from Pathlib!", encoding="utf-8")
content_pathlib = Path("hello_pathlib.txt").read_text(encoding="utf-8")
print(content_pathlib)
```
</PythonEditor>

### 2.2 處理 JSON 文件

<PythonEditor title="JSON 文件處理對比" compare={true} canRun={false}>
```javascript !! js
// JavaScript
const data = { name: "Alice", age: 30 };

// 寫入 JSON 文件
fs.writeFileSync('data.json', JSON.stringify(data, null, 2));

// 讀取 JSON 文件
const rawData = fs.readFileSync('data.json');
const parsedData = JSON.parse(rawData);
console.log(parsedData.name); // Alice
```

```python !! py
# Python
import json

data = {"name": "Alice", "age": 30}

# 寫入 JSON 文件
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, indent=2)

# 讀取 JSON 文件
with open("data.json", "r", encoding="utf-8") as f:
    parsed_data = json.load(f)
    print(parsed_data["name"])  # Alice
```
</PythonEditor>

## 3. 正則表達式

Python 的 `re` 模塊提供了強大的正則表達式功能。

<PythonEditor title="正則表達式對比" compare={true}>
```javascript !! js
// JavaScript
const text = "My email is test@example.com";
const regex = /\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b/;
const match = text.match(regex);
if (match) {
  console.log(match[0]); // test@example.com
}

// 替換
const newText = text.replace(/email/g, 'address');
console.log(newText); // My address is test@example.com
```

```python !! py
# Python
import re

text = "My email is test@example.com"
regex = r"\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b"
match = re.search(regex, text);
if match:
    print(match.group(0))  # test@example.com

# 替換
new_text = re.sub(r"email", "address", text)
print(new_text)  # My address is test@example.com
```
</PythonEditor>

## 4. Pandas: Python 數據分析利器

`pandas` 是 Python 數據科學生態系統的核心庫，它提供了 `DataFrame` 和 `Series` 兩種強大的數據結構，使得數據清洗、轉換、分析和可視化變得異常簡單。

### 4.1 Series: 一維數據

`Series` 類似於一個帶標籤的數組。

<PythonEditor title="Pandas Series 基礎" compare={true}>
```javascript !! js
// JavaScript (類似數組)
const data = [10, 20, 30, 40];
data.forEach(d => console.log(d));

const filteredData = data.filter(d => d > 20);
console.log(filteredData); // [30, 40]
```

```python !! py
# Python (Pandas Series)
import pandas as pd

data = pd.Series([10, 20, 30, 40])
print(data)

filtered_data = data[data > 20]
print(filtered_data)
```
</PythonEditor>

### 4.2 DataFrame: 二維數據

`DataFrame` 是 `pandas` 的核心，它是一個二維表格型數據結構，可以看作是 `Series` 的容器。

<PythonEditor title="Pandas DataFrame 基礎" compare={true}>
```javascript !! js
// JavaScript (數組對象)
const users = [
  { name: 'Alice', age: 30, city: 'New York' },
  { name: 'Bob', age: 25, city: 'Paris' },
  { name: 'Charlie', age: 35, city: 'London' },
];

// 篩選
const youngUsers = users.filter(u => u.age < 30);
console.log(youngUsers);

// 添加新列
users.forEach(u => u.is_senior = u.age > 30);
console.log(users);
```

```python !! py
# Python (Pandas DataFrame)
import pandas as pd

data = {
    'name': ['Alice', 'Bob', 'Charlie'],
    'age': [30, 25, 35],
    'city': ['New York', 'Paris', 'London']
}
df = pd.DataFrame(data)

# 篩選
young_users = df[df['age'] < 30]
print(young_users)

# 添加新列
df['is_senior'] = df['age'] > 30
print(df)
```
</PythonEditor>

### 4.3 從 CSV 讀取數據

`pandas` 可以非常方便地從各種數據源（CSV, Excel, SQL 等）讀取數據。

<PythonEditor title="從 CSV 讀取數據對比" compare={true} canRun={false}>
```javascript !! js
// JavaScript (Node.js with csv-parser)
const fs = require('fs');
const csv = require('csv-parser');

const results = [];

fs.createReadStream('data.csv')
  .pipe(csv())
  .on('data', (data) => results.push(data))
  .on('end', () => {
    console.log(results);
  });
```

```python !! py
# Python (Pandas)
import pandas as pd

# 假設有一個 data.csv 文件
# name,age,city
# Alice,30,New York
# Bob,25,Paris

df = pd.read_csv("data.csv")
print(df)

# 對於非常大的文件，可以分塊讀取以節省內存
# chunk_iter = pd.read_csv("large_data.csv", chunksize=1000)
# for chunk in chunk_iter:
#     process(chunk)

# 數據分析示例
print("\nAverage age:", df["age"].mean())
print("\nUsers from New York:")
print(df[df["city"] == "New York"])
```
</PythonEditor>
