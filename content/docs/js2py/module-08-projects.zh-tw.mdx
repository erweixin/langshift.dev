## 2. 項目一：URL 縮短服務

### 項目描述

構建一個功能齊全的 URL 縮短服務，類似於 `bit.ly`。用戶可以提交一個長 URL，服務會返回一個短 URL。當訪問短 URL 時，應重定向到原始的長 URL。

**核心功能**

*   使用 FastAPI 構建 API 端點。
*   使用 Pydantic 進行數據驗證。
*   使用 SQLite 或其他數據庫存儲 URL 映射。
*   生成唯一的短 URL 標識符。
*   實現重定向邏輯。

```javascript !! js
// JavaScript (Express + nanoid)
const express = require('express');
const { nanoid } = require('nanoid');

const app = express();
app.use(express.json());

const urlDatabase = {}; // 簡單內存存儲

app.post('/shorten', (req, res) => {
  const { longUrl } = req.body;
  const shortId = nanoid(7);
  urlDatabase[shortId] = longUrl;
  res.json({ shortUrl: `http://localhost:3000/${shortId}` });
});

app.get('/:shortId', (req, res) => {
  const longUrl = urlDatabase[req.params.shortId];
  if (longUrl) {
    res.redirect(longUrl);
  } else {
    res.status(404).send('Not Found');
  }
});

app.listen(3000);
```

## 3. 項目二：命令行天氣應用

### 項目描述

創建一個命令行工具，用戶可以輸入城市名稱，程序將從天氣 API (如 OpenWeatherMap) 獲取並顯示當前的天氣信息。

**核心功能**

*   使用 `argparse` 或 `typer` 解析命令行參數。
*   使用 `httpx` 或 `requests` 調用第三方 API。
*   處理 API 響應 (JSON)。
*   在終端中格式化並顯示天氣信息。

<PythonEditor title="命令行天氣應用：Python 實現思路" compare={true} canRun={false} height={900}>
```javascript !! js
// JavaScript (Node.js + axios + yargs)
const axios = require('axios');
const yargs = require('yargs/yargs');
const { hideBin } = require('yargs/helpers');

const argv = yargs(hideBin(process.argv)).argv;

async function getWeather(city) {
  try {
    const apiKey = 'YOUR_API_KEY';
    const url = `https://api.openweathermap.org/data/2.5/weather?q=${city}&appid=${apiKey}`;
    const response = await axios.get(url);
    console.log(`Weather in ${city}:`);
    console.log(`- Temp: ${response.data.main.temp}°K`);
    console.log(`- Description: ${response.data.weather[0].description}`);
  } catch (error) {
    console.error('Error fetching weather data');
  }
}

if (argv.city) {
  getWeather(argv.city);
}
```

```python !! py
# Python (Typer + httpx)
import typer
import httpx

API_KEY = "YOUR_API_KEY"

def main(city: str):
    """獲取並顯示指定城市的天氣信息"""
    url = f"https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}"
    try:
        with httpx.Client() as client:
            response = client.get(url)
            response.raise_for_status()
            data = response.json()
            print(f"Weather in {city}:")
            print(f"- Temp: {data['main']['temp']}°K")
            print(f"- Description: {data['weather'][0]['description']}")
    except httpx.HTTPStatusError as e:
        print(f"Error: Unable to fetch weather data for {city}. {e.response.status_code}")
    except Exception as e:
        print(f"An error occurred: {e}")

if __name__ == "__main__":
    typer.run(main)

# 運行: python weather_app.py London
```

## 4. 項目三：數據分析與可視化報告

### 項目描述

從一個 CSV 數據集（例如，關於銷售、電影或任何你感興趣的主題）中提取、分析數據，並生成一個簡單的 HTML 報告，其中包含關鍵指標和圖表。

**核心功能**

*   使用 `pandas` 讀取和清洗 CSV 數據。
*   進行數據聚合和分析（例如，計算平均值、總和、分組）。
*   使用 `matplotlib` 或 `seaborn` 創建圖表（如條形圖、折線圖）。
*   將圖表保存為圖片。
*   使用 `jinja2` 模板引擎生成 HTML 報告。

```python !! py
# Python (Pandas + Matplotlib + Jinja2)
import pandas as pd
import matplotlib.pyplot as plt
from jinja2 import Environment, FileSystemLoader

# 1. 數據分析
df = pd.read_csv("sales_data.csv")
total_sales = df["sales"].sum()
top_products = df.groupby("product")["sales"].sum().nlargest(5)

# 2. 創建圖表
plt.figure()
top_products.plot(kind="bar")
plt.title("Top 5 Products by Sales")
plt.savefig("top_products.png")

# 3. 生成 HTML 報告
env = Environment(loader=FileSystemLoader('.'))
template = env.get_template("report_template.html")

html_content = template.render(
    total_sales=total_sales,
    top_products_table=top_products.to_html()
)

with open("report.html", "w") as f:
    f.write(html_content)

# report_template.html (Jinja2 模板)
# <!DOCTYPE html>

#   {{ top_products_table|safe }}
```
