## 為什麼學習 Kotlin？

Kotlin 是一種現代、簡潔、安全的程式語言，運行在 JVM 上。它是 Android 開發的首選語言，並為伺服器端開發、Web 開發和跨平台應用提供出色的支援。

### 核心優勢

- **Android 開發**: Android 開發的官方語言
- **協程**: 強大的協程非同步程式設計
- **函數式程式設計**: 豐富的函數式程式設計特性
- **JVM 生態系統**: 完全存取 Java 庫和框架
- **跨平台**: Kotlin Multiplatform 共享程式碼
- **型別安全**: 空安全和強型別系統
- **互操作性**: 無縫的 Java 互操作

## 現代 Kotlin 特性

- **協程**: 簡化的非同步程式設計
- **Flow**: 響應式流資料處理
- **Jetpack Compose**: 現代宣告式 UI
- **Kotlin Multiplatform**: 跨平台程式碼共享
- **智慧轉換**: 自動型別轉換
- **擴充函數**: 擴展現有類別
- **資料類別**: 簡潔的資料容器

```kotlin !! kt
// Kotlin
val message = "Hello, World!"
println(message)

fun greet(name: String): String {
    return "Hello, $name!"
}

val result = greet("Kotlin")
println(result)
```
