# Kotlin 语言介绍

欢迎来到 JavaScript → Kotlin 学习之旅！本模块将帮助你了解 Kotlin 语言的设计哲学、生态系统，以及如何从 JavaScript 开发者的角度快速掌握这门现代编程语言。

## 什么是 Kotlin？

Kotlin 是由 JetBrains 开发的一种现代、简洁、安全的编程语言，运行在 Java 虚拟机（JVM）上。它被 Google 选为 Android 开发的官方语言，同时支持服务器端开发、Web 开发和跨平台应用开发。

### Kotlin 的核心特性

- **空安全**: 编译时检查空值，避免运行时空指针异常
- **协程**: 强大的异步编程支持，比 JavaScript 的 Promise 更灵活
- **函数式编程**: 丰富的函数式编程特性
- **互操作性**: 与 Java 100% 互操作
- **跨平台**: Kotlin Multiplatform 支持多平台开发
- **简洁语法**: 比 Java 更简洁，比 JavaScript 更安全

## 语言对比示例

让我们通过一个简单的例子来感受两种语言的差异：

<UniversalEditor title="Hello World 对比" compare={true}>
```javascript !! js
// JavaScript
let message = "Hello, World!";
console.log(message);

function greet(name) {
    return `Hello, ${name}!`;
}

const result = greet("Kotlin");
console.log(result);
```

```kotlin !! kt
// Kotlin
val message = "Hello, World!"
println(message)

fun greet(name: String): String {
    return "Hello, $name!"
}

val result = greet("Kotlin")
println(result)
```
</UniversalEditor>

### 主要差异

1. **变量声明**: JavaScript 使用 `let/const`，Kotlin 使用 `val/var`
2. **类型注解**: Kotlin 需要显式类型声明
3. **字符串模板**: Kotlin 使用 `$` 而不是 `${}`
4. **函数声明**: Kotlin 使用 `fun` 关键字
5. **输出函数**: Kotlin 使用 `println()` 而不是 `console.log()`

## 协程：Kotlin 的杀手锏

协程是 Kotlin 最强大的特性之一，让我们对比一下：

<UniversalEditor title="异步编程对比" compare={true}>
```javascript !! js
// JavaScript Promise
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}

fetchData().then(data => {
    console.log('Data:', data);
});
```

```kotlin !! kt
// Kotlin 协程
suspend fun fetchData(): String {
    return try {
        val response = URL("https://api.example.com/data").readText()
        response
    } catch (error: Exception) {
        println("Error: $error")
        ""
    }
}

// 启动协程
runBlocking {
    val data = fetchData()
    println("Data: $data")
}
```
</UniversalEditor>

### 协程的优势
- **更轻量**: 比线程更轻量，可以创建数千个协程
- **更简单**: 代码看起来像同步代码，但实际是异步执行
- **更灵活**: 支持取消、超时、异常处理等高级特性
